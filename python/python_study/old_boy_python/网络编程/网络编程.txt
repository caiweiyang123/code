4.1 物理层负责发送电信号
一组物理层数据称之为：位
单纯的电信号毫无意义，必须对其进行分组

4.2 数据链路层：ethernet以太网协议
规定一：一组数据称之为一个帧
规定二：数据帧分成两部分=》头+数据
        头包含：源地址与目标地址，改地址是mac地址
        数据包含的：是网络层整体的内容
规定三：规定单反接入互联网的主机必须有一块网卡，每块网卡在出厂的时候都会烧制好一个全世界独一无二的地址，mac地址

注意：计算机通信基本靠吼，即以太网的工作方式：广播

4.3 网络层：IP协议
划分广播域
每一个广播域但凡要与外界通信，一定要有一个网关帮计算机转发包到公网
网关与外界通信走的是路由协议

规定一：一组数据称之为一个数据包
规定二：数据帧分成两部分=》头+数据
    头包含：源地址与目标地址，改地址是IP地址
    数据包含：传输层整体的内容

4.3.1 总结：
ip地址+mac地址可以标识全世界范围内独一无二的计算机

4.4传输层 tcp/udp =》基于端口
    端口范围：0-65535，0-1023 为系统占用端口
    ip+port=>标识全世界范围内独一无二的一个基于网络通信的应用

    基于tcp协议通信之前，必须建立一个双向通信的链接
    C-------------->S
    C<--------------S
    三次握手建立连接：
        建立连接是为了传输数据准备的，三次握手即可

    四次挥手断开连接：
        断开连接时，由于连接内有数据传输，所以必须分四次断开
    tcp是可靠传输的
        发送数据必须等到对方确认后才算完成，才会将自己内存中的数据清空
    ps：当服务端大量处于TIME_WAIT状态时意味着服务端正在经历高并发
